# Memory Consolidation

The process by which short-term memories become long-term memories. Happens primarily during sleep, especially REM sleep. The hippocampus replays experiences and strengthens connections to the neocortex.

Three stages:
1. Encoding: initial formation in working memory
2. Consolidation: transfer to long-term storage during sleep
3. Retrieval: reactivation through [[cognitive/spreading-activation]]

The [[systems/rem-agent]] is a direct biomimicry of this process:
- Decay = forgetting curve (Ebbinghaus). Unused connections weaken.
- Prune = clearing irrelevant memories during sleep
- Dream = discovering new connections between existing memories

This is why Shomo named it the REM Agent. During biological REM sleep, the brain does exactly what the dream operator does: finds unexpected connections between recently encoded experiences and existing knowledge.

Related: [[cognitive/priming]] (retrieval strengthens consolidation), [[cognitive/chunking]] (how we organize what gets consolidated).

The two-process model (Diekelmann and Born, 2010) distinguishes active consolidation (hippocampal replay) from synaptic homeostasis (pruning). The REM agent implements both.
